# Projet Bataille Navale

## Développeur
- Nom : Oussama FILALI
- Email : oussfil@gmail.com

## Description
Implémentation d'un jeu de Bataille Navale en Python avec interface graphique Tkinter.

## Fonctionnalités
- Interface graphique avec Tkinter
- Mode joueur contre ordinateur
- Placement manuel des navires pour le joueur
- Placement automatique pour l'ordinateur
- Système de tour par tour
- Indication visuelle des tirs (touchés/manqués)
- Affichage des navires coulés

## Structure du Projet
Le projet est développé en plusieurs phases :

### Phase 1 : Structure de Base ✓
- Mise en place des classes fondamentales
- Configuration initiale du projet

### Phase 2 : Interface Utilisateur ✓
- Création des grilles de jeu
- Mise en place des éléments graphiques

### Phase 3 : Placement des Navires ✓
- Système de placement manuel
- Algorithme de placement automatique

### Phase 4 : Logique de Jeu ✓
- Système de tir
- Gestion des tours
- Détection de fin de partie

### Phase 5 : Améliorations ✓
- Optimisations
- Corrections de bugs
- Améliorations visuelles
